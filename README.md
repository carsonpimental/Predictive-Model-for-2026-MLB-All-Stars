# Predictive-Model-for-2026-MLB-All-Stars
Built supervised classification models (Logistic Regression, Decision Trees, Random Forest, and Naïve Bayes) using historical MLB All-Star selections and 2026 Steamer projections to forecast 2026 All-Star Game candidates, using a custom evaluation metric that improves recall–precision tradeoffs relative to F1 for All-Star candidate identification.
